## 📖 عن المشروع

**مدونة العرائش التقنية** هي منصة عربية متخصصة تهدف لنشر المعرفة التقنية وتطوير المحتوى العربي. تحتوي المدونة على أكثر من **1000 مقال حصري** وأكثر من **500 فيديو تعليمي** في مجالات متعددة.

### ✨ المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 📱 **تصميم متجاوب** | يعمل بشكل مثالي على جميع الأجهزة (جوال، جهاز لوحي، حاسوب) |
| 🌙 **الوضع المظلم التلقائي** | يتكيف مع تفضيلات نظام المستخدم |
| 🔍 **بحث متقدم** | بحث فوري مع اقتراحات ذكية |
| 🏷️ **تصنيفات متعددة** | تنظيم المحتوى في أقسام متخصصة |
| 📄 **تحميل تدريجي** | عرض المقالات بشكل متدرج لتحسين الأداء |
| 🔗 **تحسين SEO متكامل** | دعم كامل لـ Open Graph و Twitter Card |
| 🗺️ **خريطة موقع ديناميكية** | عرض أحدث المقالات بشكل منظم |
| 📧 **نشرة بريدية** | نموذج اشتراك لجذب الزوار |
